## Flujo de Datos - Ejemplo: Crear Objetivo

```
1. Frontend (React)
   в”Ӯ
   в”ңв”Җв–ә POST /api/goals
   в”Ӯ   Content-Type: application/json
   в”Ӯ   Body: {
   в”Ӯ     "title": "Mi objetivo",
   в”Ӯ     "category": "daily",
   в”Ӯ     "priority": "high"
   в”Ӯ   }
   в”Ӯ
2. API Layer (main.py вҶ’ app/api/routes/goals.py)
   в”Ӯ
   в”ңв”Җв–ә @router.post("/")
   в”Ӯ   в”ңв”Җв–ә Recibe GoalCreate (Pydantic schema)
   в”Ӯ   в”ңв”Җв–ә Valida automГЎticamente
   в”Ӯ   в”ңв”Җв–ә Inyecta dependencia: db = Depends(get_db)
   в”Ӯ
3. Service Layer (app/services/goal_service.py)
   в”Ӯ
   в”ңв”Җв–ә GoalService.create_goal(db, goal)
   в”Ӯ   в”ңв”Җв–ә Genera UUID Гәnico
   в”Ӯ   в”ңв”Җв–ә Crea instancia Goal (SQLAlchemy model)
   в”Ӯ   в”ңв”Җв–ә db.add(goal) y db.commit()
   в”Ӯ
4. Database Layer (app/db/database.py)
   в”Ӯ
   в”ңв”Җв–ә SQLAlchemy engine
   в”Ӯ   в”ңв”Җв–ә Convierte a SQL INSERT
   в”Ӯ   в”ңв”Җв–ә Ejecuta con connection pool
   в”Ӯ
5. SQL Server
   в”Ӯ
   в”ңв”Җв–ә INSERT INTO goals (id, title, category, priority, ...)
   в”Ӯ   VALUES ('uuid', 'Mi objetivo', 'daily', 'high', ...)
   в”Ӯ
6. Response
   в”Ӯ
   в”ңв”Җв–ә Goal instance del DB
   в”Ӯ   в”ңв”Җв–ә Serializa a GoalResponse (Pydantic)
   в”Ӯ   в”ңв”Җв–ә Convierte a JSON
   в”Ӯ
7. Frontend (React)
   в”Ӯ
   в””в”Җв–ә Recibe 200 OK + JSON con Goal creado
```

## Flujos de Negocio Principales

### 1. Flow: Crear Objetivo con Subobjetivos

```
POST /api/goals
в””в”Җв–ә Goal creado

POST /api/goals/{goal_id}/subgoals
в””в”Җв–ә SubGoal creado (relacionado)

GET /api/goals/{goal_id}
в””в”Җв–ә Goal con lista de subgoals
```

### 2. Flow: SesiГіn Diaria de Preguntas

```
GET /api/daily-sessions/2024-02-19
в””в”Җв–ә SesiГіn para hoy (o crea nueva)
    в”ңв”Җ total_questions: 5
    в”ңв”Җ answered_questions: 0
    в””в”Җ responses: []

POST /api/daily-sessions/2024-02-19/responses
в”ңв”Җ Body: [{question_id, response}, ...]
в””в”Җв–ә SesiГіn actualizada
    в”ңв”Җ answered_questions: 5
    в””в”Җ responses: [...]
```

### 3. Flow: Frases Inspiracionales

```
GET /api/phrases/categories
в””в”Җв–ә [MotivaciГіn, Productividad, ...]

GET /api/phrases?category_id=abc
в””в”Җв–ә [Frases en esa categorГӯa]

POST /api/phrases/{phrase_id}/review
в””в”Җв–ә review_count +1, last_reviewed_at = now
```
